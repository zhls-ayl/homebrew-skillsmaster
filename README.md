# homebrew-skillsmaster

Homebrew tap for installing SkillsMaster.

## Install

```bash
brew tap zhls-ayl/skillsmaster
brew install --cask skillsmaster
```

## Upgrade

```bash
brew update
brew upgrade --cask skillsmaster
```
